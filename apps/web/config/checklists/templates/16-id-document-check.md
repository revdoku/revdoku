ID Document Check

Review a scanned or photographed identity document — passport, driver's licence, national ID, or similar — for the essentials a valid ID needs to show.

- Document type (Passport, Driver's Licence, National ID, etc.) is identifiable
- Full name is visible and readable
- Date of birth is visible and plausible (not in the future, not unreasonably old)
- Document number is visible and in the expected format for the document type
- Issue date is visible and not in the future
- Expiry date is visible and not in the past
- Issuing country or jurisdiction is visible
- Photo of the bearer is present and not obviously tampered with
- Required security features visible in the scan (hologram reflection, UV marks, MRZ line) are present where legible
- The document is not obviously cropped, redacted, or watermarked "SPECIMEN" / "VOID"
