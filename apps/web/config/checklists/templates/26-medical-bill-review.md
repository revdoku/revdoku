Medical Bill Review

Review a medical billing statement for common billing errors including duplicate charges, unbundled procedures, incorrect codes, and unreasonable quantities.

If a CPT / HCPCS / ICD-10 code reference document is attached, cross-check every code on the billing statement against the reference and flag any code not found there.

- No duplicate charges: each CPT / HCPCS code appears only once per date of service unless clinically justified (e.g., bilateral procedure with modifier 50)
- Procedures are not unbundled: when a comprehensive surgical procedure is billed (e.g., 29881 surgical arthroscopy), component procedures (e.g., 29870 diagnostic arthroscopy) should not be billed separately on the same date per NCCI edits
- Each CPT / HCPCS code correctly matches its written description — for example, injection / IV codes should not be used for oral medications
- Anesthesia time is reasonable for the type of procedure (routine knee arthroscopy typically 45–60 minutes total, not 90+)
- The date of service is consistent across all line items for a single-session procedure
- The provider NPI is present and correctly formatted (10 digits)
- The subtotal equals the sum of all line item charges (verify the math)
- The patient responsibility equals subtotal minus insurance adjustment minus insurance paid
- Surgical supplies and DME charges are reasonable and not duplicated
