# Presentation Files

This bucket is a starter file collection with two sample PDF presentation decks
and a floating feedback form.

It intentionally does not include `index.html`. When published, Revdoku will
serve the generated Auto-Index Page so visitors can browse and preview the PDF
files. The template metadata registers a hosted `feedback` form, so Revdoku
injects a floating widget where visitors can leave comments without this bucket
needing its own form page.
