# AI Shared Workspace

This bucket is private shared storage for collaborating with AI agents. Agents
can review, organize, analyze, and update the files here without making them
public.

Start with `project-brief.md`, keep agent-facing context in `agent-handoff.md`,
and use the sample CSV files for analysis or testing:

- `sales-by-quarter.csv` contains quarterly revenue and customer totals.
- `sales-pipeline.csv` contains example opportunities and next steps.

Add source files alongside them as needed. Nothing in this private bucket is
published or publicly accessible unless its owner explicitly publishes it.
