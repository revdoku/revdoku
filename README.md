# REVDOKU

Publish websites from Claude, ChatGPT, Codex using Revdoku.
Few seconds from idea to a live website that you can share!

```sh
revdoku --publish ./dist
revdoku --publish --protected --generate-password ./dist
```

Protected websites are available on Pro plans. Open the returned URL and enter
the copied password; do not pass passwords in URL parameters.
